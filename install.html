Plugin Comment

Pour CakePHP
Présentation Installation Utilisation Configuration

La première chose à faire est de copier le plugin Comment dans le dossier Plugin de votre application. Pour celà vous pouver télécharger le .zip ou cloner directement le dépôt

git clone git@github.com:Grafikart/CakePHP-Comment.git Comment

Ensuite il vous faut charger le plugin, pour cela dans votre fichier bootstrap.php vous pouvez décommenter la ligne

CakePlugin::loadAll(array(
        'Comment' => array('bootstrap' => true)
      ));

Voila ! le plugin est bien chargé vous pouvez maintenant importer la base de donnée qui va permettre de gérer les différents médias. Pour cela il faut utiliser la console de CakePHP et utiliser la commande

cake schema create --plugin=Comment

Et c'est tout vous êtes prêt à utiliser le plugin !
