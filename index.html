Plugin Comment

Pour CakePHP
Présentation Installation Utilisation Configuration

Le principe de ce plugin CakePHP est de vous permettre de créer un système de commentaire rapidement et simplement. Le système repose sur un behaviour à greffer sur les model que l'on souhaite rendre commentable.

Fonctionnalités

  • Association hasMany Comment pour vos model
  • Possibilité d'utiliser les sessions utilisateur
  • Possibilité d'avoir des commentaire "threaded", on peut répondre à un commentaire
  • Intégration du filtre antiSpam Akismet

Compatible avec CakePHP 2.1.X
