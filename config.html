Plugin Comment

Pour CakePHP
Présentation Installation Utilisation Configuration

Bootstrap

Le plugin peut être configurer en modifiant une ou plusieurs valeur dans Configure::read('Plugin.Comment') : 

array(
  // Contient les infos à utiliser si un utilisateur est loggué
  'user'   => array(
    'model' => 'User',                // Model contenant les utilisateurs
    'username' => 'username',         // Champ à utiliser comme pseudo
    'mail'     => 'mail',             // Champ à utiliser comme mail
  ),
  'order'  => 'Comment.created ASC',  // Ordre d'organisations de commentaires
  'subcomments' => true               // On autorise les réponses à un commentaire

  // Optionnel
  'akismet'=> array('site' => SITE,'key' => KEY),  // Si vous souhaiter utiliser akismet il faut entrer l'url de votre site ainsi que votre clef

);
